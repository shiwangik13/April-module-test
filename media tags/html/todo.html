Block Element

Block elements are those which starts from a new line

Examples:

  1. h1 to h6 tag
  2. p tag
  3. div tag
  4. form tag
  5. ul tag
  6. ol tag

Inline Element

Inline elements which start from the point where the previous element ended.

Examples:

  1. a tag
  2. span tag
  3. img tag
  4. br tag
  5. button tag
  6. input tag
  7. label tag
